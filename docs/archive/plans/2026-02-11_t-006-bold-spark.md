## 4. Key Decisions

1. **Self-hosted fonts** over Google Fonts CDN — better performance, privacy, consistent with project patterns
2. **Geometric CSS art** over placeholder images in hero — no external dependencies, scales to any viewport
3. **Inline stats** in BoldHome rather than Content Collection — same pattern as Minimal Zen, avoids scope creep
4. **Skip View Transitions** — will be addressed in T-010 across all themes
5. **Full design spec** — implemented all diagonal sections, offset shadows, animated counters, parallax effects

---

## 5. Key Discoveries

- Astro scoped styles + JS-toggled visibility requires inline styles (confirmed from T-005)
- `clip-path: polygon()` creates distinctive diagonal section transitions that define the Bold Spark aesthetic
- Offset box-shadows (`8px 8px 0 var(--color-primary)`) on cards give a strong graphic/print feel
- IntersectionObserver stats counter animation adds significant energy to the page
- Spring cubic-bezier easing (`cubic-bezier(0.34, 1.56, 0.64, 1)`) creates bouncy, energetic hover effects
