## 1. Goal

Implement the 5th and final CleanSpark theme — Noir Luxe — a premium dark theme with charcoal background, champagne gold accents, Playfair Display serif headings, Lato light body text, film grain overlay, and luxury aesthetics. All 6 pages fully styled and mobile responsive.

## 2. Approach

Followed established multi-theme architecture: independent components per theme, scoped styles with class prefixes, CSS custom properties from theme config, IntersectionObserver scroll-reveal, and static import map in page-resolver.

### Design Decisions (from user input)
- Hero art: Gold diamond/gem CSS art (rotated square, inner border, radial glow, orbiting ring, gold dots)
- Film grain: SVG feTurbulence noise overlay at 3.5% opacity
- Stats section: Gold styling with counter animation
- Gallery: Grayscale-to-color hover effect (60% grayscale default)

### Noir-Specific Conventions
- Typography: Playfair Display 400/600 (headings), Lato 300/400 (body)
- Buttons: Uppercase, 0.15em letter-spacing, outlined gold that fills on hover
- Cards: Dark surface (#1A1A1A), thin #2A2A2A borders, gold border glow on hover
- Scroll animations: Slower, more elegant (0.8s ease vs other themes' 0.6s)
- Section labels: Lato 300 weight, 0.15em tracking, gold color
- Class prefixes: `.nn` (nav), `.nf` (footer)

## 4. Key Discoveries

- Film grain as inline SVG data URL (feTurbulence filter) avoids external file dependency and works reliably across browsers
- Split-centered nav (logo center, links split left/right) creates a distinct luxury feel different from all other themes
- Grayscale-to-color hover on gallery images is a simple but effective premium touch (just CSS filter transition)
- Gold rule draw-in animation (`scaleX(0) → scaleX(1)`) defined once in layout, reused across all pages via `.gold-rule` class
- SVG icons for values section (instead of emoji) maintain the premium aesthetic better than Bubbly's emoji approach
